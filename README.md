# TEST TRANSKBANK

Se desarrolló la aplicación con gradle para compilación de dependencias, se anexa collection para prueba en local.
Así mismo de debe realizar la configuración del gradle en local y ejecutar gradle bootRun para iniciar la apliciación por el puerto 8089.
Una vez levantada la aplicación importar el collection anexo a postman y realizar las pruebas respectivas.

# ESPECIFICACIONES

El poryecto está realizado en java 1.8 con lombok como banco de datos en memoria, persistencia de datos y springboot.
